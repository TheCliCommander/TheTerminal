I'm writing this guide assuming you have a basic understanding of networking in general. A basic understanding of the OSI model is suggested. Check out my page [placeholder] to learn the essentials.

Discovering your private IP address:

To discover your private IP address you can use the ip command. Note: it may still be popular to use the ifconfig command but this is deprecated and should be replaced with the ip command. You should understand that your machine has what is called a 'Network Interface Controller'. This device is what communicates with your LAN and the internet. It will have a name like eth0 or wlan0(the latter of which is your wireless or 'WiFi' interface, while the former is your wired ethernet connection). These devices have two identifying addresses. One is a unique hardcoded address called a link layer or media access control/MAC address. The other is called the Internet Protocol address, or IP address. We'll talk about the MAC address in another section.


To use the ip command to discover your current ipv4 address use the following syntax ip -4 a. For your ipv6 address: ip -6 a. To get both ipv4 and ipv6 just use ip a. Really simple right?
